# 📋 모두의 연수등록부 (Everyone's Training Registry)

> **무설정 단일 HTML 파일 기반 (Standalone Portable Web App)**  
> 학교 연수·협의회·회의 등의 출석체크 및 정보 수집을 디지털화하는 반응형 웹 애플리케이션입니다.

웹 서버, Node.js, Vite 등의 빌드 환경이 전혀 필요 없으며, **`index.html` 파일을 로컬 컴퓨터에서 더블 클릭(file:// 프로토콜)하는 것만으로 완벽히 작동**합니다.

---

## ✨ 핵심 기능

1. **단일 HTML 패키징 (Zero-Setup Portable)**
   - 자바스크립트 비즈니스 로직, HSL 다크 테마 디자인 시스템 CSS가 단 하나의 `index.html` 파일 내부에 통합되어 있습니다.
   - USB에 담거나 메신저로 전달하여 로컬 브라우저에서 바로 더블 클릭해 실행할 수 있습니다.
2. **동적 폼 빌더 (Form Builder)**
   - 출석체크 양식에 기입받을 추가 수집 필드를 설계할 수 있습니다.
   - 단답형, 장문형, 수치, 단일선택, 다중선택, 날짜/시간, 안내창, 구분선 모듈 지원.
3. **프리미엄 자필 서명 패드 (Signature Pad)**
   - Pointer Events 통합 제어로 모바일 터치 및 PC 마우스 서명을 모두 제공합니다.
   - 부드러운 베지어 곡선 보간과 속도 가속에 따른 잉크 두께 모사(필압) 기술을 구현했습니다.
   - 스캔받은 서명 이미지 파일(PNG/JPG)을 불러와 서명란에 렌더링하는 대체 수단도 기본 제공합니다.
4. **출석 대장 및 파일 내보내기**
   - **CSV 다운로드**: 엑셀 한글 깨짐을 방지하는 UTF-8 BOM 처리가 기본 적용되어 있습니다.
   - **PDF 출력**: 나눔고딕 한글 서체를 CDN에서 동적으로 로드해 virtual VFS에 등록하여 한글 깨짐을 해결하고, 참석자의 자필 서명을 표 내에 정밀 임베딩하여 문서를 생성합니다.
5. **로컬 Mock DB 지원**
   - 구글 연동 없이도 브라우저 `localStorage`를 데이터베이스 삼아 등록, 관리자 제어, 설계 등 전 기능을 로컬 테스트해 볼 수 있습니다.

## 🚀 사용법

### 1. 로컬에서 실행하기 (기본 작동)
- [index.html](file:///f:/안티그래비티/모두의 연수등록부/index.html) 파일을 마우스로 **더블 클릭**하여 브라우저로 엽니다.
- 초기 관리자 페이지 비밀번호는 **`1234`** 입니다.

### 2. 구글 스프레드시트(클라우드 DB) 연동하기
공유 링크를 통해 실제 동료들의 실시간 출석 현황을 모으려면 아래 절차를 통해 구글 시트와 웹앱을 결합하세요.

1. [Google Drive](https://drive.google.com)에서 새 스프레드시트를 생성합니다.
2. 상단 메뉴의 **확장 프로그램** -> **Apps Script**를 선택합니다.
3. 프로젝트 내의 `gas/Code.gs` 소스코드를 복사하여 Apps Script 에디터에 덮어쓰고 저장합니다.
4. 우측 상단 **배포** -> **새 배포**를 누른 뒤 유형을 **웹 앱**으로 지정합니다.
   - *웹앱을 실행할 사용자*: `나 (본인 계정)`
   - *액세스 권한이 있는 사용자*: **`모든 사용자`** (참석자가 비로그인 상태로도 접속해 제출할 수 있어야 함)
5. 배포 후 발급된 **웹 앱 URL**을 복사합니다.
6. `index.html`을 실행한 뒤 **관리 대시보드** 하단 연동 설정란에 해당 주소를 입력하고 **저장**을 누르면 연동이 개시됩니다.
